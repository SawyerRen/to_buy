## 用户

### 登录

Url: http://127.0.0.1:8000/login

Method: POST

Params:

```python
email     		str # 邮箱
phone_number	str # 电话号码
password		str # 密码
```

Return:

```json
{
    "id": 2,
    "is_deleted": false,
    "created_at": "2022-03-27T22:22:04Z",
    "updated_at": "2022-03-28T09:55:47Z",
    "name": "rsy",
    "password": "123456",
    "phone_number": "111",
    "email": "test2@gmail.com",
    "gender": 1,
    "membership_expire_time": null,
    "membership": 1
}
```

### 注册前检查有没有重复的电话号码或邮箱

Url: http://127.0.0.1:8000/register_check?email=test1@gmail.com&phone_number

Method: GET

Params:

```python
email     		str # 邮箱
phone_number	str # 电话号码
```

Return:

```json
{
    "email": "test1@gmail.com",
    "phone_number": "",
    "count": 1
}
```

### 注册

Url: http://127.0.0.1:8000/register

Method: POST

Params:

```python
name			str # 用户名
email     		str # 邮箱
phone_number	str # 电话号码
password		str # 密码
password2     	str # 确认密码
gender			int # 性别：1男，2女，0默认
```

Return:

```json
{
    "id": 14,
    "is_deleted": false,
```
